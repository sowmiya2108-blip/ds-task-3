# ds-task-3
This task focuses on building a Decision Tree Classifier to predict whether a customer will purchase a product or service based on their demographic and behavioral data. 🎯 Objective To develop a machine learning model that: Classifies customer behavior Predicts purchase decisions Understands key factors influencing customer actions
🌳 Data Science Internship - Task 3

📌 Overview

This project is part of the Data Science Internship at Prodigy Infotech.
The objective is to build a Decision Tree Classifier to predict whether a customer will purchase a product or service based on demographic and behavioral data.

---

📂 Dataset

The dataset used is the Bank Marketing Dataset, which contains customer-related information such as:

- Age
- Job
- Marital Status
- Education
- Contact Details
- Previous Campaign Outcomes

---

🛠️ Tools and Technologies

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

---

🤖 Model Building

Steps Performed:

1. Data Preprocessing
   
   - Handled missing values
   - Encoded categorical variables
   - Prepared features and target variable

2. Model Training
   
   - Applied Decision Tree Classifier
   - Split dataset into training and testing sets

3. Model Evaluation
   
   - Checked accuracy
   - Analyzed performance of the model

---

📊 Results

The Decision Tree model successfully identified patterns in customer data and predicted whether a customer is likely to subscribe to a product/service.

---

📌 Conclusion

This project demonstrates how machine learning models like Decision Trees can be used for classification problems and decision-making based on real-world data.
